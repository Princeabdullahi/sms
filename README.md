# Golden Olives Academy Links App

This is a simple installable web app for Golden Olives Academy.

## How to edit the links
1. Open config.js.
2. Replace the placeholder URLs with your real website and social links.
3. Save the file and refresh the browser.

## Run locally
From the project folder, run:

```bash
npm start
```

Then open http://localhost:8000

## Make it an app
This project is already set up as a Progressive Web App (PWA), so it can be installed on phones and desktops.

### Android / iPhone
- Open the site in a browser.
- Use the browser's Install or Add to Home Screen option.

### Optional native packaging
If you want a true app store app later, you can wrap this web app with Capacitor or Cordova.

Example:

```bash
npx cap init
npx cap add android
npx cap sync
```

## App logo
Replace the placeholder logo in assets/icon.svg with your real GOA logo when you have it.
